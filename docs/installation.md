# Tempo Installation

1. Download the Tempo OS image from: [FIXME](fixme)
2. Install the Raspberry Pi Imager from the [official download page](https://www.raspberrypi.com/software/)
3. Insert the target SD Card into your computer and launch the Raspberry Pi Imager software
    - Raspberry Pi Device: Choose Raspberry Pi Zero 2 W
    - Operating System: Scroll all the way to the bottom and select 'Use Custom'
    - Select the Tempo OS image you downloaded earlier
    - Storage: Select the target SD Card

4. After it completes writing, eject the SD Card andinsert it into the Raspberry Pi Zero 2 W.
5. Power on the Raspberry Pi with the Inky Impression display connected.
6. Follow the on-screen instructions to complete the Wi-Fi setup and initial configuration.
7. Once wifi is complete, from your computer connect to http://tempo.local in your web browser to access the Tempo web interface and configure your calendar.
